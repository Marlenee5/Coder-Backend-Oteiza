# API de Productos y Carritos - BackEnd1

## Descripción
Proyecto de Backend con Node.js y Express que implementa una API REST para gestionar **productos** y **carritos de compra**.  
La persistencia de datos se realiza mediante archivos JSON (`products.json` y `carts.json`), por lo que no se requiere base de datos.
